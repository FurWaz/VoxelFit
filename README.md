# VoxelFit

Minigame about matching voxel shapes, inspired by Replicube on Steam.

Created using :
- [Nuxt Framework](https://nuxt.com/)
- [Three.js](https://threejs.org/)

### Note
Color constants are :
```
RED: 0xfb2c36
ORANGE: 0xff6900
YELLOW: 0xefb100
GREEN: 0x00c951
CYAN: 0x00b8db
BLUE: 0x2b7fff
PURPLE: 0xad46ff
PINK: 0xf6339a
WHITE: 0xfafafa
BLACK: 0x09090b
GREY: 0x71717b
GRAY: 0x71717b
```

## Setup

Make sure to install dependencies:

```bash
# npm
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev
```

## Production

Build the application for production:

```bash
# npm
npm run build
```

Locally preview production build:

```bash
# npm
npm run preview
```
